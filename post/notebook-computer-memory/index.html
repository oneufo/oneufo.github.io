ASUS超极本电脑内存升级记录

2019-12-08
#Memory 

前几天安装 Catalina 10.15 后，运行程序操作起来感觉有点缓慢，决定将电脑原来的4G内存升级为8G。

通过官方客服电话联系到当地售后点咨询了一下，再多加一条4G内存需要240RMB。

然后去搜索了相关资料发现笔记本4G内存只需要149，接着开始搜索看一些拆机换内存的教程。

最后决定自己购买安装，这样也可以锻炼自己的动手能力。

通过硬件检测软件确定型号后在Tmall官方旗舰店下单购买。

安装成功后续更新……
